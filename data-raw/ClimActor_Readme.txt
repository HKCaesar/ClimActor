DATA & FILE OVERVIEW

1. File List: 
Files are always designated with MM and YYYY in the file name.
- country_dict_August2020.csv - country-level dictionary for data cleaning, includes sociodemographic information (land area, GDP, per World Bank statistics).
- key_dict_Sept2020.csv - key dictionary for data cleaning functions
- subnational_contextuals_database_Sept2020.csv - database of subnational climate actors with relevant contextual information.
- subnational_contextuals_database_initcols_Sept2020.csv - database of subnational climate actors with separate binary columns (0 = no participation; 1 = participation) for climate action networks participation.
